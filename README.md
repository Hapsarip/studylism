## Getting Started
Setting up project for local usage.
1. Clone or download this repository
```shell
https://github.com/Hapsarip/studylism.git
```
2. Install NPM Pacages
```shell
npm install
```
3. Make .env file in server folder.
```shell
RESTSTUDYLISM_DB_URI="your mongo string connection"
JWT_SECRET="your jwt key"
```
4. Makes sure that you're in the server folder, then run
```shell
npm start
```
